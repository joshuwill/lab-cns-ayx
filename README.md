# lab-cns-ayx

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存储，可以通过仓库页面直接访问或部署到静态托管服务。

## 项目简介

这是一个用于存放 HTML 页面的项目仓库，所包含的页面均为独立开发或收集整理。每个页面都有其特定的功能或展示内容，不依赖于特定后端服务。仓库本身仅作为页面存档与分发用途。

## 目录说明

```
/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 页面归档目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/` 目录存放所有独立 HTML 页面文件。
- `assets/` 目录存放页面所依赖的公共资源文件。
- 根目录下的 `index.html` 可作为仓库的导航或说明页面（视情况而定）。

## 页面归档说明

每个 HTML 页面均为独立、自包含的静态文件。页面之间互不依赖，可通过直接打开文件或在服务器上部署后访问。归档页面可能包含但不限于以下类型：

- 信息展示页面
- 工具页面
- 实验性页面
- 其他可静态呈现的内容

页面可能使用外部资源（如 CDN 加载的库），但不会依赖特定域名或网络环境。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 下的任意 HTML 文件。
3. 或将整个仓库目录部署到任意静态托管服务，通过 URL 访问页面。
